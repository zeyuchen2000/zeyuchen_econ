"Land Titling Reform and Spatial Redistribution: Mitigating Effects of Urban Construction Land Allocation"
Zeyu Chen
Working paper (January 2024)
  • This link provides a zip file containing data and Python codes for solving the spatial equilibrium model and conducting counterfactual simulation: [Python codes (2024.01.06)].
  • I know Matlab usually handles matrix calculations better, but I still prefer using Python because of personal habit. Here are two replication exercises where I translated Matlab code into Python, and these exercises also taught me a lot about how to solve a spatial equilibrium model:
    • The first exercise is from Caliendo and Parro's (2015, RES) paper tilted "Estimates of the Trade and Welfare Effects of NAFTA." Their original Matlab code is available on the RES paper webpage. I translated their Matlab scripts from the "Counterfactuals" and "Equilibrium" folders into Python, and the replicated code is provided in a zip file here: [Python codes (2023.02.11)]
    • The second exercise replicates a spatial economics paper from a top Chinese journal, titled "Spatial Allocation of Land and New Development Pattern—Based on the Analysis of Quantitative Spatial Equilibrium" (《土地的区域间配置与新发展格局——基于量化空间均衡的研究》). The authors' original Matlab codes are available here, and my Python replication is provided in this zip file: [Python codes (2023.05.04)].
